IA’m — Intégration

Où placer votre badge IA’m

```

Le badge IA’m se place au plus proche du contenu qu’il décrit. Il doit être visible au moment où le contenu est découvert.

Article / page web

Placez le badge en bas de l’article ou en introduction.

Image / création

Ajoutez le badge directement sur l’image ou dans sa légende.

PDF / document

Placez le badge en pied de page ou en dernière page.

Réseaux sociaux

Ajoutez le badge dans le visuel ou la description.

Si le badge n’est pas visible sans le chercher, il est mal placé.
```
