Mode d’emploi

Comment utiliser le badge IA’m

Le badge IA’m se place au plus proche du contenu qu’il décrit. Il doit être visible au moment où le contenu est découvert.

01

Article / page web

Placez le badge en bas de l’article, dans l’introduction ou près des crédits du contenu.

02

Image / création

Ajoutez le badge directement sur le visuel ou dans la légende qui accompagne l’œuvre.

03

PDF / document

Insérez le badge en pied de page, dans les crédits ou sur la dernière page du document.

04

Réseaux sociaux

Placez le badge dans l’image publiée ou mentionnez le niveau IA’m dans la description.

La règle simple

Le badge accompagne le contenu.

IA’m n’est pas un logo global à cacher dans un footer. C’est une indication contextuelle, attachée à une œuvre, un texte, une image ou un document précis.

Si le lecteur ne voit pas le badge sans le chercher, il est probablement trop loin du contenu.
